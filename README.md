# olegkrotov.ru

Portfolio website built with [Astro](https://astro.build/) and deployed to GitHub Pages.

## Development

Requires Node.js 22.12 or newer and pnpm.

```sh
pnpm install
pnpm dev
```

Production build:

```sh
pnpm build
pnpm preview
```

Every push to `main` triggers the GitHub Pages deployment workflow.
